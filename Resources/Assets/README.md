# Assets

App-native visual assets for historical forms, modern forms, icons, and typography.

For the symbol evolution experience, Origin, Oracle Bone, Bronze, Seal, Clerical, and Regular stages should use dedicated image assets or explicit missing-asset placeholders. Do not reuse the modern regular character as a fake historical stage.

The character-evolution reference images are product targets for composition and navigation, but they are not production assets unless licensing and source quality are explicitly resolved.
